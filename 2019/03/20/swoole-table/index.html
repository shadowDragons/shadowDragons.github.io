「Swoole」Table 内存表

Posted by Ben Blog on March 20, 2019

swoole_table一个基于共享内存和锁实现的超高性能，并发数据结构。用于解决多进程/多线程数据共享和同步加锁问题。
基于行锁，所以单次set/get/del在多线程/多进程的环境下是安全的。
set/get/del等方法是原子操作，用户代码中不需要担心数据加锁和同步的问题。

优势

  • 性能强悍，单线程每秒可读写200万次
  • 应用代码无需加锁，swoole_table内置行锁自旋锁，所有操作均是多线程/多进程安全。用户层完全不需要考虑数据同步问题。
  • 支持多进程，swoole_table可以用于多进程之间共享数据
  • 使用行锁，而不是全局锁，仅当2个进程在同一CPU时间，并发读取同一条数据才会进行发生抢锁

接口

构造 __construct
function Table->__construct(int $size, float $conflict_proportion = 0.2)
参数 说明
$size 参数指定表格的最大行数，如果$size不是为2的N次方，如1024、8192,65536等，底层会自动调整为接近的一个数字，如果小于1024则默认成1024，即1024是最小值
$conflict_proportion 是哈希冲突的最大比例，超过这个比例，共享内存表就不允许再添加新的行元素
增加列 column

内存表增加一列

bool Table->column(string $name, int $type, int $size = 0);
参数 说明
$name 指定字段的名称
$type 指定字段类型，支持3种类型，Table::TYPE_INT, Table::TYPE_FLOAT, Table::TYPE_STRING
$size 指定字符串字段的最大长度，单位为字节。字符串类型的字段必须指定$size
类型 说明
Table::TYPE_INT 默认为4个字节，可以设置1，2，4，8一共4种长度
Table::TYPE_STRING 设置后，设置的字符串不能超过此长度
Table::TYPE_FLOAT 会占用8个字节的内存
创建 create

创建内存表。
执行create向操作系统申请内存

function Table->create() : bool;
参数 说明
返回 系统内存不足，申请失败返回false，反之，返回true

Table使用共享内存来保存数据，在创建子进程前，务必要执行create

Server中使用Table，create必须在Server->start()前执行

获取实际占用内存的尺寸使用$table->memorySize

设置值 set

设置行的数据，Table使用k-v的方式来访问数据。

Table->set(string $key, array $value) : void
参数 说明
$key 数据的key，相同的$key对应同一行数据，如果set同一个key，会覆盖上一次的数据
$value 必须是一个数组，必须与字段定义的$name完全相同

如果传入字符串长度超过了列定义时设定的最大尺寸，底层会自动截断。

自增 incr

原子自增操作。

function Table->incr(string $key, string $column, mixed $incrby = 1) : int;
参数 说明
$key 指定数据的key，如果$key对应的行不存在，默认列的值为0
$column 指定列名，仅支持浮点型和整型字段
$incrby 增量，默认为1。如果列为整形，$incrby必须为int型，如果列为浮点型，$incrby必须为float类型
返回 返回最终的结果数值
自减 decr

原子自减操作。

function Table->decr(string $key, string $column, mixed $decrby = 1) int | float;
参数 说明
$key 指定数据的key，如果$key对应的行不存在，底层会首先将该行数据初始化为0
$column 指定列名，仅支持浮点型和整型字段
$decrby 减量，默认为1。如果列为整形，$decrby必须为int型，如果列为浮点型，$decrby必须为float类型
返回 返回最终的结果数值
获取值 get

获取一行数据

array Table->get(string $key, string $field = null);
参数 说明
$key 指定查询数据行的KEY，必须为字符串类型
$field 当设置该值仅返回该字段的值，而不是整个记录
返回 key不存在，返回false，反之，返回结果
key是否存在 exist

检查table中是否存在某一个key。

bool swoole_table->exist(string $key);
参数 说明
$key 指定查询数据行的KEY，必须为字符串类型
返回 不存在，返回false，反之，返回true
获取总数 count

返回table中存在的条目数

int function Table->count();
删除 del

删除数据

bool Table->del(string $key)
参数 说明
$key 指定查询数据行的KEY，必须为字符串类型
返回 key不存在，返回false，反之，返回true

实例

数组
<?php
$table = new swoole_table(1024);
$table->column('id', swoole_table::TYPE_INT);
$table->column('name', swoole_table::TYPE_STRING, 64);
$table->column('num', swoole_table::TYPE_FLOAT);
$table->create();

$table['apple'] = array('id' => 145, 'name' => 'iPhone', 'num' => 3.1415);
$table['google'] = array('id' => 358, 'name' => "AlphaGo", 'num' => 3.1415);

$table['microsoft']['name'] = "Windows";
$table['microsoft']['num'] = '1997.03';

var_dump($table['apple']);
var_dump($table['microsoft']);

$table['google']['num'] = 500.90;
var_dump($table['google']);
web-socket
<?php
$table = new swoole_table(1024);
$table->column('fd', swoole_table::TYPE_INT);
$table->column('from_id', swoole_table::TYPE_INT);
$table->column('data', swoole_table::TYPE_STRING, 64);
$table->create();

$serv = new swoole_server('127.0.0.1', 9501);
$serv->set(['dispatch_mode' => 1]);
$serv->table = $table;

$serv->on('connect', function($serv, $fd, $from_id){
	$info = $serv->connection_info($fd);
	$serv->send($fd, "INFO: fd=$fd, from_id=$from_id, addr={$info['remote_ip']}:{$info['remote_port']}\n");
});

$serv->on('receive', function ($serv, $fd, $from_id, $data) {

	$cmd = explode(" ", trim($data));

	//get
	if ($cmd[0] == 'get')
	{
		//get self
		if (count($cmd) < 2)
		{
			$cmd[1] = $fd;
		}
		$get_fd = intval($cmd[1]);
		$info = $serv->table->get($get_fd);
		$serv->send($fd, var_export($info, true)."\n");
	}
	//set
	elseif ($cmd[0] == 'set')
	{
		$ret = $serv->table->set($fd, array('from_id' => $data, 'fd' => $fd, 'data' => $cmd[1]));
		if ($ret === false)
		{
			$serv->send($fd, "ERROR\n");
		}
		else
		{
			$serv->send($fd, "OK\n");
		}
	}
	else
	{
		$serv->send($fd, "command error.\n");
	}
});

$serv->start();
