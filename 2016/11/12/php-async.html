php实现异步

November 12, 2016

php在处理异步执行主要有以下四种方式来处理，其中第四种方式是最好的，推荐使用。

1.ajax或嵌入img标签，在src中指向异步地址。
缺点 不能算是真正异步，主要用户关闭浏览器，异步脚本也就被终止掉了

<html>
<head>
</head>
<body>
<img src="b.php"> 
<script src="jquery.js"></script>
<script>
$(document).ready(function(){
    $.ajax({url:"b.php",async:false});
});    
</script>
</body>
</html>

2.popen
缺点 只能在命令行中执行本地脚本，如果访问量高，会产生大量进程

pclose(popen("php -f /Users/zhongjunbin/Document/Kitematic/nginx/htdocs/b.php &", 'r'));

3.curl
缺点 会有至少一秒的等待时间

$ch = curl_init();
$curl_opt = array(CURLOPT_URL, 'http://www.example.com/backend.php',
                            CURLOPT_RETURNTRANSFER, 1,
                            CURLOPT_TIMEOUT, 1,);
curl_setopt_array($ch, $curl_opt);

4.fsockopen 或 socker_stream_client
缺点 需要自己拼接http头信息

<?php
$fp = fsockopen("192.168.99.100", 80, $errno, $errstr, 30);
if (!$fp) {
    echo "$errstr ($errno)<br />\n";
} else {
    $out = "GET /b.php HTTP/1.1\r\n";
    $out .= "Host: 192.168.99.100\r\n";
    $out .= "Connection: Close\r\n\r\n";
    fwrite($fp, $out);
    /*忽略执行结果
    while (!feof($fp)) {
        echo fgets($fp, 128);
    }*/
    fclose($fp);
}

<?php
$fp = stream_socket_client("192.168.99.100:80", $errno, $errstr, 30);
if (!$fp) {
    echo "$errstr ($errno)<br />\n";
} else {
    fwrite($fp, "GET /b.php HTTP/1.0\r\nHost: 192.168.99.100\r\nAccept: */*\r\n\r\n");
    /*忽略执行结果
    while (!feof($fp)) {
         echo fgets($fp, 1024);
    }*/
    fclose($fp);
}
